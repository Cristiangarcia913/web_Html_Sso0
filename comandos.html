Terminal: Conceptos Básicos

La terminal (o línea de comandos) es una interfaz de texto para interactuar con el sistema operativo. Aunque las interfaces gráficas son dominantes, la terminal ofrece mayor control y eficiencia para muchas tareas. Los sistemas operativos modernos incluyen terminales avanzadas con características como:

  • Autocompletado con la tecla Tab
  • Historial de comandos (usando las flechas arriba/abajo)
  • Redirección de entrada/salida con >, <, |
  • Scripting para automatizar tareas
bash
user@host:~$ ls -l
total 12
drwxr-xr-x 2 user user 4096 Jan 10 10:00 Documents
drwxr-xr-x 4 user user 4096 Jan 12 15:30 Projects
-rw-r--r-- 1 user user 123 Jan 15 09:45 notes.txt
user@host:~$ _

Comparativa de Comandos

Navegación de Archivos

  • ls -la - Listar archivos incluyendo ocultos
  • cd ~ - Ir al directorio home
  • pwd - Mostrar ruta actual
  • tree - Mostrar estructura de directorios

Gestión de Archivos

  • cp -r dir1 dir2 - Copiar directorios recursivamente
  • mv archivo1 archivo2 - Renombrar archivos
  • rm -rf directorio - Eliminar directorio y contenido
  • chmod 755 archivo - Cambiar permisos

Procesos y Sistema

  • top - Monitorizar procesos en tiempo real
  • ps aux | grep proceso - Buscar procesos
  • kill -9 PID - Terminar proceso por ID
  • df -h - Ver espacio en disco

Redes y Conexiones

  • ifconfig - Configuración de red
  • ping dominio.com - Probar conectividad
  • ssh usuario@servidor - Conexión remota
  • scp archivo usuario@servidor:ruta - Copia segura

Navegación de Archivos

  • dir /a - Listar archivos incluyendo ocultos
  • cd %userprofile% - Ir al directorio usuario
  • tree /f - Mostrar estructura de directorios

Gestión de Archivos

  • copy /y archivo1 archivo2 - Copiar sin confirmar
  • ren archivo1 archivo2 - Renombrar archivos
  • del /s /q directorio - Eliminar recursivamente

Sistema y Redes

  • tasklist - Listar procesos
  • taskkill /pid 1234 /f - Terminar proceso
  • ipconfig /all - Configuración de red
  • netstat -ano - Conexiones de red

Administración

  • systeminfo - Información del sistema
  • chkdsk /f - Reparar disco
  • sfc /scannow - Reparar archivos sistema

Comandos Especiales

  • open . - Abrir Finder en ubicación actual
  • say "hola" - Conversión texto a voz
  • brew install paquete - Instalar con Homebrew
  • diskutil list - Listar dispositivos de almacenamiento

Redes y Seguridad

  • networksetup -listallhardwareports - Interfaces de red
  • security find-generic-password -wa "nombre" - Buscar en llavero
  • codesign -dv --verbose=4 /aplicacion.app - Verificar firma apps

Sistema

  • system_profiler SPHardwareDataType - Especificaciones hardware
  • softwareupdate -l - Listar actualizaciones disponibles
  • pmset -g - Configuración energía

Trucos Avanzados

Redirección y Pipes

Combinar comandos es una de las características más potentes:

# Redirigir salida a archivo
ls -l > listado.txt

# Redirigir errores
comando 2> errores.log

# Pipe para filtrar resultados
ps aux | grep chrome | sort -k3 -r

Historial y Atajos

Optimiza tu flujo de trabajo con estos trucos:

# Buscar en historial (Ctrl+R)
(reverse-i-search)`comando`

# Ejecutar último comando que empiece con...
!ls

# Sustituir texto en último comando
^original^reemplazo

Atajos útiles: Ctrl+A (inicio línea), Ctrl+E (fin línea), Ctrl+U (borrar línea)

Scripting Básico

Crea scripts para automatizar tareas:

#!/bin/bash
# Script de backup
fecha=$(date +%Y%m%d)
tar -czf backup_$fecha.tar.gz /ruta/a/respaldar
echo "Backup completado el $(date)" >> log.txt

Ejecutar con: chmod +x script.sh y luego ./script.sh

Redes Avanzadas

Diagnóstico y manejo de redes:

# Escanear puertos abiertos
nmap -sV IP

# Monitorizar tráfico (Linux)
tcpdump -i eth0 -n

# Conexión persistente SSH
ssh -o ServerAliveInterval=60 user@host
