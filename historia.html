Línea de Tiempo de los Sistemas Operativos

1940-1950

Primera Generación

Sistemas sin SO. Programación directa en lenguaje máquina.

1955-1965

Sistemas Batch

GM-NAA I/O (1956) considerado el primer SO. Ejecución de trabajos por lotes.

1965-1980

Tercera Generación

Multiprogramación, time-sharing. Sistemas como UNIX (1969), OS/360.

1980-1990

Computación Personal

MS-DOS (1981), Mac OS (1984). Interfaz gráfica de usuario.

1990-2000

SO Modernos

Windows 95, Linux (1991), sistemas distribuidos.

2000-Presente

Era Móvil y Cloud

Android, iOS, Chrome OS. Computación en la nube y contenedores.

Hitos Importantes

1969 - UNIX

Desarrollado por Bell Labs. Base para muchos SO modernos.

1985 - Windows 1.0

Primera versión del sistema operativo gráfico de Microsoft.

1991 - Linux

Linus Torvalds lanza el kernel Linux bajo licencia GPL.

2007 - iOS

Apple revoluciona los SO móviles con el iPhone.
