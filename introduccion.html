¿Qué es un Sistema Operativo?

Un sistema operativo (SO) es el software principal que gestiona los recursos hardware y provee servicios comunes para los programas de aplicación. Actúa como intermediario entre el usuario y el hardware del computador.

Definición Formal

Según Tanenbaum (2015): "Un sistema operativo es un programa que controla la ejecución de programas de aplicación y actúa como interfaz entre las aplicaciones y el hardware del computador".
